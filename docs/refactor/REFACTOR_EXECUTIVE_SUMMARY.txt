DIRECT FUNCTIONS IMPLEMENTED:

User Management:

  • updateUserDirect() - Rename/update user role/permissions
  • deleteUserDirect() - Remove user from config

Service Management:

  • disableServiceDirect() - Disable service

  • Session state monitoring and reconciliation
